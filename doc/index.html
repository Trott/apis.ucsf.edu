APIs.ucsf.edu JavaScript Base Object

JavaScript APIs hosted at APIs.ucsf.edu reside in an object in the global namespace called UCSF.

The base object may be loaded in a web page like so:

<script src="http://apis.ucsf.edu/jsapi"></script>

In practice, however, you would almost never load the base object like that. Instead, you would request all the other code required for your APIs at the same time. For example, you could get the base object (UCSF) and the Person Search API code (UCSF.Person) like so:

<script src="http://apis.ucsf.edu/jsapi?person"></script>

Or if you wanted to get the Shuttle API code as well (UCSF.Shuttle), you could get it all with one HTTP request:

<script src="http://apis.ucsf.edu/jsapi?person&amp;shuttle"></script>

There is some basic scaffolding inside the UCSF object for CORS/XHR support.
